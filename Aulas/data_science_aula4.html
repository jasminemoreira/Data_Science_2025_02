Tendência Central

Onde os Dados se Concentram

Média, Mediana e Moda

📍 Conceito

Medidas de tendência central indicam o valor típico ou representativo de um conjunto de dados, servindo como ponto de referência para análises.

🎯 Média Aritmética

x = Σx / n

Quando usar: Dados simétricos, sem outliers extremos
Vantagem: Usa todos os valores
Desvantagem: Sensível a outliers

🎪 Mediana

Definição: Valor central quando dados ordenados

Quando usar: Dados assimétricos, presença de outliers
Vantagem: Robusta a outliers
Desvantagem: Ignora valores extremos

📊 Moda

Definição: Valor mais frequente

Quando usar: Dados categóricos, identificar padrões
Vantagem: Representa valor mais comum
Desvantagem: Pode não existir ou ser múltipla

⚖️ Escolhendo a Medida Adequada

Distribuição Melhor Medida Razão
Normal/Simétrica Média Média = Mediana = Moda
Assimétrica Mediana Não afetada pela assimetria
Com Outliers Mediana Robusta a valores extremos
Categórica Moda Única medida aplicável

Dispersão

Variabilidade dos Dados

Amplitude, Variância, Desvio Padrão e Quartis

📐 Conceito

Medidas de dispersão quantificam o quanto os dados se afastam da tendência central, indicando homogeneidade ou heterogeneidade do conjunto.

📏 Amplitude

Amplitude = Máximo - Mínimo

Vantagem: Simples e intuitiva
Desvantagem: Considera apenas extremos

📊 Variância

σ² = Σ(x - μ)² / N
s² = Σ(x - x)² / (n-1)

Vantagem: Usa todos os valores
Desvantagem: Unidade quadrática

📐 Desvio Padrão

σ = √σ²
s = √s²

Vantagem: Mesma unidade dos dados
Interpretação: Dispersão típica em relação à média

📦 Quartis e IQR

IQR = Q3 - Q1

Q1: 25% dos dados
Q3: 75% dos dados
IQR: Amplitude interquartil (robusto)

🎯 Coeficiente de Variação

CV = (σ / μ) × 100%

Medida relativa de dispersão, permite comparar variabilidade entre conjuntos com unidades ou magnitudes diferentes.

📊 Interpretação da Dispersão

  • Baixa dispersão: Dados homogêneos, previsíveis
  • Alta dispersão: Dados heterogêneos, maior incerteza
  • Regra 68-95-99.7: Para distribuição normal
  • Outliers: Valores além de 1.5×IQR dos quartis

Controle de Qualidade: Duas máquinas produzem parafusos com diâmetro médio de 5mm. Máquina A: σ=0.1mm, Máquina B: σ=0.5mm. Qual escolher?

Investimentos: Como usar desvio padrão para avaliar risco de carteiras?

Relacionamentos

Correlação e Covariância

Medindo relacionamentos entre variáveis

🔗 Conceito

Correlação e covariância medem o grau e direção do relacionamento linear entre duas variáveis, fundamentais para análise multivariada e machine learning.

📈 Covariância

Cov(X,Y) = Σ(x - x)(y - y) / (n-1)

Interpretação:
• Positiva: Variáveis crescem juntas
• Negativa: Uma cresce, outra decresce
• Zero: Sem relacionamento linear

🎯 Correlação de Pearson

r = Cov(X,Y) / (σₓ × σᵧ)

Vantagem: Padronizada (-1 a +1)
Interpretação:
• |r| > 0.7: Correlação forte
• 0.3 < |r| < 0.7: Moderada
• |r| < 0.3: Fraca

📊 Tipos de Correlação

  • Linear: Pearson (paramétrica)
  • Monotônica: Spearman (não-paramétrica)
  • Não-linear: Kendall Tau
  • Categórica: Cramér's V, Phi

⚠️ Cuidados Importantes

  • Correlação ≠ Causalidade
  • Outliers afetam correlação
  • Relações não-lineares podem ser perdidas
  • Variáveis confundidoras

🎪 Matriz de Correlação

Ferramenta essencial para análise multivariada, mostra correlações entre todas as variáveis simultaneamente. Útil para:

  • Identificar multicolinearidade
  • Seleção de features
  • Análise exploratória
  • Validação de modelos

💭 Aplicações em Data Science

Feature Selection: Duas variáveis com correlação r=0.95. O que isso implica para machine learning?

Análise de Negócio: Correlação entre gastos em marketing e vendas é 0.85. Como interpretar e usar essa informação?

Causalidade: Sorvete e afogamentos têm correlação positiva. Há causalidade?

Validação

Testes de Hipótese

Validação estatística de suposições

🧪 Conceito

Testes de hipótese são procedimentos estatísticos para decidir se há evidência suficiente nos dados para apoiar ou rejeitar uma afirmação sobre a população.

🎯 Elementos Fundamentais

H₀ (Hipótese Nula): Afirmação a ser testada
H₁ (Hipótese Alternativa): Afirmação contrária
α (Nível de Significância): Probabilidade de erro tipo I
p-valor: Probabilidade dos dados sob H₀

⚖️ Tipos de Erro

Erro Tipo I (α): Rejeitar H₀ verdadeira
Erro Tipo II (β): Aceitar H₀ falsa
Poder do Teste (1-β): Detectar efeito real
Trade-off: ↓α pode ↑β

📊 Testes Comuns

Teste Objetivo Dados
Teste t (uma amostra) Média = valor específico Quantitativos, normais
Teste t (duas amostras) Comparar duas médias Dois grupos independentes
ANOVA Comparar múltiplas médias Múltiplos grupos
Chi-quadrado Independência/associação Categóricos
Shapiro-Wilk Normalidade Qualquer distribuição

🔍 Processo de Teste

  • Formular H₀ e H₁
  • Escolher nível α (usualmente 0.05)
  • Calcular estatística de teste
  • Determinar p-valor
  • Decisão: p < α → rejeitar H₀

⚠️ Interpretação Cuidadosa

  • p < α: Evidência contra H₀
  • p ≥ α: Sem evidência suficiente
  • Nunca "provar" H₀
  • Significância ≠ importância prática

💭 Aplicações Práticas

A/B Testing: Nova versão do site aumenta conversão? Como estruturar o teste de hipótese?

Controle de Qualidade: Processo produtivo está dentro do padrão? H₀: μ = 50kg, α = 0.01

Exemplo

Case: Análise de Performance de Vendas

Aplicando todas as medidas estatísticas

1. Contexto

E-commerce analisando performance de vendedores: 50 vendedores, 6 meses de dados, vendas mensais e tempo de resposta

2. Análise Descritiva

Vendas: média R$ 45.000, mediana R$ 42.000, σ = R$ 12.000. Tempo resposta: média 2.3h, mediana 1.8h

3. Análise de Correlação

Correlação vendas × tempo resposta: r = -0.68 (moderada negativa). Menor tempo = maiores vendas

4. Teste de Hipótese

H₀: μ_vendas = R$ 40.000 vs H₁: μ_vendas > R$ 40.000. Resultado: p = 0.023 < 0.05 → rejeitar H₀

📊 Descobertas

  • Performance média acima da meta
  • Distribuição assimétrica (alguns super-performers)
  • Tempo de resposta é fator crítico
  • 15% dos vendedores são outliers superiores

💡 Insights Acionáveis

  • Treinar equipe em agilidade de resposta
  • Benchmark dos top performers
  • Meta individual baseada em quartis
  • Sistema de alertas para outliers negativos

🎯 Aplicação das Medidas

Aplicação das Medidas
Mediana: Meta realista (robusta a super-performers)
Desvio Padrão: Identificar variabilidade excessiva
Correlação: Direcionar treinamentos
Teste t: Validar se metas foram atingidas

# Estatísticas descritivas

📊 Python - scipy.stats

from scipy import stats

# Teste t
stats.ttest_1samp(data, 50)

# Normalidade
stats.shapiro(data)

# Correlação
stats.pearsonr(x, y)

🗃️ SQL

SELECT
AVG(vendas) as media,
STDDEV(vendas) as desvio,
PERCENTILE_CONT(0.5)
WITHIN GROUP (ORDER BY vendas)
as mediana
FROM vendas_mes

📈 Visualização - seaborn

import seaborn as sns

# Correlação
